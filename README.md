# sales-outreach

A personal workspace for AI-assisted B2B sales outreach, built around the [Andrej Karpathy-inspired Claude guidelines](https://github.com/forrestchang/andrej-karpathy-skills).

## Purpose

Improve the quality and reply rate of enterprise outreach by giving AI assistants (Claude, Cursor, Perplexity, etc.) a consistent set of rules for:

- Sequence writing and rewrites
- Persona-specific messaging
- Prospect research
- Lightweight outreach automation

## Context

- **Role:** Enterprise sales at Noesis (part of Grupo Altia)
- **Focus:** OutSystems low-code platform, digital transformation, app delivery
- **Verticals:** Healthcare, pharma, financial services
- **Primary tool:** Apollo.io (sequences, email, prospecting)
- **Secondary channels:** LinkedIn, phone, Loom video

## Files

| File | Purpose |
|------|---------|
| `CLAUDE.md` | Behavioral guidelines for any AI assistant working in this repo |
| `PERSONAS.md` | Target buyer personas and their priorities *(coming soon)* |
| `PROMPT_LIBRARY.md` | Reusable prompts
